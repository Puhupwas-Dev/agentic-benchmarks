# Contributing to this Awesome List

Thank you for your interest! Please read the [AGENTS.md](AGENTS.md) guide thoroughly before making a contribution.

## How to Add an Entry

1. Fork the repository.
2. Find the appropriate category section in `README.md`.
3. Insert your link in the correct alphabetical order (case‑insensitive).
4. Use the format:  
   `- [Project Name](https://example.com) — Short description ending with a period.`
5. Ensure the description is concise (one sentence) and explains why the project is awesome.
6. Run `awesome-lint README.md` locally to catch formatting errors.
7. Open a pull request with a clear title (e.g., `Add ProjectName to SectionX`).

## What We Accept

- Actively maintained projects (updated within the last 12 months unless historically significant)
- High‑quality, authoritative resources
- Clear relevance to the [POWERSHARE] space

## What We Reject

- Dead links or abandoned projects
- Random link dumps without explanation
- Duplicate entries
- Overly promotional language

## Questions?

Open an issue and label it `question`. Thank you for helping keep this list curated and useful!
